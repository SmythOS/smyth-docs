# Sign Component

The Sign component in `SmythOS` generates digital signatures using HMAC or RSA methods.

Choose the signature generation method:

1. **HMAC:** Combines a cryptographic hash function with a secret key to produce a message authentication code.
2. **RSA:** Uses asymmetric key encryption with a valid RSA private key (e.g., PEM, PKCS#1 format) to create digital signatures.

> **Note:** The private key signs the message for RSA, while the public key verifies it.
